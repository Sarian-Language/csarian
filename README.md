# Sarian (CSarian) – Version 0.1 (Alpha)

**Sarian** is an interpreted programming language written in C, designed for simplicity and rapid development. This is the **first alpha version (0.1)** and still under active development, so you can expect unfinished code and experimental features.

## Installation

You can install Sarian in two ways:

### 1. Using precompiled releases
Check the [GitHub releases](https://github.com/Sarian-Lang/csarian) for downloadable binaries for your platform.

### 2. Compiling from source
```bash
git clone https://github.com/Sarian-Lang/csarian.git
cd csarian
make
````

This will compile the interpreter to `build/sarian`.
